## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить звёздочкой (\*), например так:

* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты пронумеровать, например так:

1. Первый пункт
2. Второй пункт
